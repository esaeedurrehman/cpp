# variable
- A variable is used to store data.
- The value of a variable can change during program execution
 

## Example
int age = 18;  
-int → Data type  
-age → Name of Variable  
-18 → Value of Variable

# Comments
- Comments are used to explain the code.
- The compiler ignores comments.
- Comments do not affect program execution.

## Example
// This is a single-line comment  
/* This is  
a multi-line  
comment */

# Low-level laungage
- Low-level language is close to machine language.
- It is fast and efficient.
- It is difficult for humans to understand.
- Process in number form.

## Example
- Machine Language
- Assembly Language

# High-level laungage
- High-level language is easy for humans to understand.
- It is easy to write and maintain.
- It requires a compiler or interpreter.

## Examples:
- C++
- Python
- Java
- JavaScript 

C++ is considered a middle-level language because it has features of both high-level and low-level languages.


# Data Types
- A data type tells the compiler what type of data should be stored in a variable.

# Types of Data Types
## 1.int
int = Integer
- int is used to store whole numbers.
## Example:
- int age = 18;
- int marks = 95;

## 2.float
float = Floating Point
- float is used to store decimal numbers.
## Example:
- float pi = 3.14;
- float price = 99.99;

## 3.char
char = character
- char is used to store a single character.
## Example:
- char grade = 'A';
- char ch = 'd';

## 4. double
double = Double Precision Floating Point
- double is used to store large decimal numbers with more precision.
## Example
- double salary = 12345.6789;

## 5. bool
bool   = Boolean
- bool is used to store true or false values.
## Example
- bool isPassed = true;
