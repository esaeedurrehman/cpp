# Structure of CPP

```cpp
#include <iostream>

int  main () {
    std::cout <<"Hello world!"<<endl;

    return 0;
}
```

## \#include \<iostream\>
- this line says there is a file named `iostream` already.
- Bring all important things out of that file.
- The file is called header file
- There exists many other header files
- Header files enhance program functionalities
- Syntax is `#include` and then file name in angular brackets
- in above program, only iostream is included
### iostream
- help with input/output
- IO stands for input/output
- other functionalities also exists
- `cout` is coming from this file

## Blank lines
- Blank lines and whitespaces are ignored by compiler
