# History and Evolution 

- C++ was developed by Bjarne Stroustrup in 1979.
- C++ is an extension of the C language.
- C++ allows programmers to write fast programs.
- It provides more control over system resources and memory management.
- C++ is known for its high performance.

## Versions of C++:
- C++11 (2011) – Major update with many modern features.
- C++14 (2014) – Improvements and bug fixes over C++11.
- C++17 (2017) – Added new features and improved performance.

# Basic Structue

```cpp
#include <iostream>

using namespace std;

int main()
{
    cout << "Hello World";
    return 0;
}
```
- `#include <iostream>` is used for input and output operations.
- `using namespace std;` allows us to use standard C++ objects without writing `std::`.
- `main()` is the entry point of a C++ program.
- `Curly braces{ }` define the body of a function.
- `cout` is used to display output on the screen.
- `return 0;` indicates successful program execution.
