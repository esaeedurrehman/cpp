# Operators
- Operators are special symbols that perform operations on variables and values.

## Arithmetic Operators
### Addition +
   - Used to add two or more values.
   - Example: 10 + 5 = 15

### Subtraction -
   - Used to subtract one value from another.
   - Example: 10 - 5 = 5

### Multiplication *
   - Used to multiply two values.
   - Example: 10 * 5 = 50

### Division /
   - Used to divide one value by another.
   - Example: 10 / 5 = 2
   - If both values are integers, the decimal part is removed.
     Example: 5 / 2 = 2

### Modulus (Remainder) % 
   - Used to find the remainder after division.
   - Example: 11 % 5 = 1



## Increment and Decrement Operators

### Increment ++ 
   - Increases the value of a variable by 1.
   - Same as: a = a + 1

### Decrement --
   - Decreases the value of a variable by 1.
   - Same as: a = a - 1


## Types of Increment/Decrement

### 1. Post Increment (a++)
   - First uses the current value.
   - Then increases the value by 1.

### 2. Pre Increment (++a)
   - First increases the value by 1.
   - Then uses the updated value.

### 3. Post Decrement (a--)
   - First uses the current value.
   - Then decreases the value by 1.

### 4. Pre Decrement (--a)
   - First decreases the value by 1.
   - Then uses the updated value.

## Assignment Operators
- Assignment operators are used to assign (store) values in variables.
### Basic Assignment Operator
=  → Assigns a value to a variable.
#### Examples:
int a = 3;
- Creates an integer variable.
- Stores the value 3 in variable a.

int b = 9;
- Creates an integer variable.
- Stores the value 9 in variable b.

char d = 'd';
- Creates a character variable.
- Stores the character 'd' in variable d.

## Comparison Operators 
- Comparison operators compare two values.
- They always return:  
   True  (1 for true)  
   False (0 for false)

### == Equal to
- Checks if both values are equal.

### != Not Equal to
- Checks if both values are different.

### >= Greater than or Equal to
- Checks if the left value is greater than or equal to the right value.

### <= Less than or Equal to
- Checks if the left value is less than or equal to the right value.

### > Greater than
- Checks if the left value is greater than the right value.

### <  Less than
- Checks if the left value is less than the right value.
